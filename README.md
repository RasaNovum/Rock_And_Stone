Rock n Stone is a mod that changes ore distribution to follow individual sets of climate and environmental conditions, making certain ores spawn in more dedicated types of terrain than in Vanilla.

Fabric builds are provided for Minecraft 1.20.1 and 1.21.1, with a NeoForge build for Minecraft 1.21.1.

Each target build also writes version-correct datapack templates to
`versions/<target>/build/templates/`.

If you have any questions that aren't [answered in our Wiki](https://github.com/RasaNovum/Rock_And_Stone/wiki), feel free to [ask on our Discord](https://discord.com/invite/WGh4mq6W5U)!

## Dependencies

- **[Fabric API](https://modrinth.com/mod/fabric-api)** - For Fabric versions
- **[NeoForge](https://neoforged.net/)** - For NeoForge version
- **[MidnightLib](https://modrinth.com/mod/midnightlib)** - Provides config
